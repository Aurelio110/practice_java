1) Welche Zugriffsmodifizierer für Felder in einem Interface sind erlaubt?
	
	b) public, static und final  
		
----------------------------------------------------------------------------------
2)  Welche Modifizierer für Methoden sind erlaubt in einem Interface?

       a) abstract, static, public und private

----------------------------------------------------------------------------------
3)  Welche Aussage zu Interfaces in Java trifft zu?
	
	c) Interfaces können andere Interfaces mittels keyword "extends" erweitern.
		
----------------------------------------------------------------------------------
4) Welche Aussage trifft zu?

	a) Eine Klasse kann beliebig viele Interfaces implementieren, aber nur von einer Klasse erben.
